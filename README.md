# FinSight - Personal Finance Analytics

Projeto desenvolvido em Python para análise financeira pessoal, com foco em inteligência financeira, projeção de fluxo de caixa e visualização de indicadores por meio de dashboards interativos.

## Objetivos

- Organizar as finanças pessoais.
- Automatizar o controle de cartões de crédito.
- Projetar o fluxo de caixa futuro.
- Construir dashboards interativos com Streamlit.
- Aplicar conceitos de análise de dados com Python.

## Funcionalidades

- Cálculo de receitas
- Cálculo de despesas
- Cálculo do saldo
- Gastos por categoria
- Top maiores despesas

## Tecnologias

- Python
- Pandas
- Matplotlib
- Streamlit
- Git e GitHub
